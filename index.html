ServerSecuree Privacy Policy

Welcome! This privacy policy explains how ServerSecuree, our anti-nuke/raid Discord bot, handles your data.

1. Information We Collect

ServerSecuree collects minimal information necessary for functionality. This includes:

  • Server IDs where the bot is installed
  • Channel IDs for logging purposes
  • Basic message events to detect raids or nukes

2. Information We Do NOT Collect

ServerSecuree does not collect:

  • User DMs or private messages
  • User personal information like email, IP addresses, or payment info
  • Any data not required for moderation and anti-raid features

3. How We Use Data

Data is used strictly to:

  • Monitor and prevent raid or nuke activity in your server
  • Log moderation events for server admins
  • Provide statistics to improve bot performance

4. Data Sharing

ServerSecuree does not share your server data with third parties. All data remains private and secure.

5. Contact

If you have questions or concerns about your data, contact the creator of this bot via Discord with the username ilyazarik.

6. Disclaimer

ServerSecuree is provided "as-is" for moderation purposes. By using this bot, you agree that:

  • We are not responsible for any server damage, data loss, or user conflicts that may occur while using the bot.
  • We do not guarantee complete prevention of raids, nukes, or other malicious activity.
  • We are not liable for actions taken by server members, administrators, or other bots in your server.
  • Users are responsible for configuring the bot correctly and following Discord's Terms of Service.
